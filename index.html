About Me

A little background

I'm Tafsir Ahmed, a Computational Engineering student at Lappeenranta University of Technology (LUT). I took the Software Development Skills — Front-End module to broaden my skills beyond engineering maths and simulations.

Over seven sessions I went from an empty text editor to a deployed, responsive website. The biggest surprise was how much you can achieve with pure CSS — from grid layouts to interactive accordions — without writing a single line of JavaScript.

Outside of coursework I'm interested in numerical methods, data visualisation, and finding ways to present technical results clearly on the web.

FAQ

Common questions about the course and project

VS Code with the Live Server extension for local development, and GitHub Pages for free hosting. No frameworks, just plain HTML and CSS.
It uses the CSS checkbox hack: a hidden checkbox paired with a label. When the label is clicked it toggles the checkbox state, and CSS sibling selectors show or hide the menu accordingly.
Flexbox works in one direction — a row or a column — and is great for navigation bars and aligning items. CSS Grid works in two dimensions at once and is ideal for card layouts and page structure.
The project files were pushed to a public GitHub repository. GitHub Pages was then enabled in the repository settings, which automatically serves the site at a public URL within minutes.
